
# Дипломный проект 
веб-сервер, который реализует функциональность планировщика задач.
Это аналог TODO-листа.

Из заданий повышенной сложности реализованы:
1. использование переменной окружения TODO_PORT
2. поиск задач по тексту или дате
3. создание докер образа

По умолчанию приложение доступно по адресу http://localhost:7540
Порт по умолчанию 7540
Можно задать переменную окружения TODO_PORT, указав другое значение порта

Для корректной работы тестов путь к БД в поле `DBFile` в файле `tests/settings.go` должен быть `"../data/scheduler.db"`
Чтобы протестировать поиск, присвойте переменной Search значение `true` в файле `tests/settings.go`

Сборка Docker образа:
`docker build -t go-scheduler .`

Запуск контейнера с портом по умолчанию (7540):
`docker run -d \
  --name scheduler \
  -p 7540:7540 \
  -v /$(pwd)/data:/app/data \
  go-scheduler`

Запуск контейнера с другим портом:
`docker run -d \
  --name scheduler \
  -p 8080:8080 \
  -v /$(pwd)/data:/app/data \
  -e TODO_PORT=8080 \
  go-scheduler`
