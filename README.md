
# Веб-сервер 
реализует функциональность планировщика задач.
Это аналог TODO-листа.
БД PostgreSQL

По умолчанию приложение доступно по адресу http://localhost:7540
Порт по умолчанию 7540

Можно задать переменную окружения TODO_PORT, указав другое значение порта

Используется Docker Compose

Собрать образы и запустить контейнеры в фоне
`docker-compose up -d`
