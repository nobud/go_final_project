
# Дипломный проект 
веб-сервер, который реализует функциональность планировщика задач.
Это аналог TODO-листа.

Из заданий повышенной сложности реализованы:
1. поиск задач по тексту или дате
2. создание докер образа

Приложение доступно по адресу http://localhost:7540

Чтобы протестировать поиск, присвойте переменной Search значение `true` в файле `tests/settings.go`

Сборка Docker образа:
`docker build -t go-scheduler .`

Запуск контейнера:
`docker run -d \
  --name scheduler \
  -p 7540:7540 \
  -v $(pwd):/app \
  go-scheduler`
